About the online version

The online version is free now, means that you can use our online mesh simplification service for free, note that the speed may vary greatly when users are using the online service simultaneously.

 

The online version is suitable for beginner indie developers, the maximum number of vertices per mesh is 65000, so you can not simplify huge 3d models with this version. If you have a huge 3d model, you have to export a low polygon version(less than 65000 vertices per mesh) with your design tool(Blender, LightWave3D, 3DS MAX, Maya etc.), then use the online version to make LOD from within Unity's editor.

 

We recommend the standalone version for Unity Free users, and the standard version for Unity Pro users, they are more productive and can simplify huge 3d models directly.

 

We reserve the rights of any changes of the free service in future.

 

Introduction

Mantis LOD Editor is an editor extensions plugin which helps you to make LOD within Unity.

Our product can generate high quality LOD version with very low number of triangles, this is very important for mobile games, no other products can achieve this.

 

Features

1. Generate LOD levels for both static meshes and skinned meshes. (All Version)

2. Protect uv boundaries and material boundaries automatically. (All Version)

3. Protect all open boundaries or smartly merge some smooth parts of open boundaries. (All Version)

4. Super fast, it can simplify 50,000 triangles per second. (Standard Version)

5. Preview final effect in real time. (Standard Version)

 

How to install?

Standard Version: Copy the subdirectories of Editor and Plugins to your project’s assets path, restart Unity Pro.

Online Version: Copy the subdirectory of Editor to your project’s assets path.

Trial Version: Unzip MantisLODEditorOnlineTrial.zip, then copy MantisLODEditorTrial.cs to the subdirectory of Editor of your project’s assets path, if the subdirectory of Editor does not exist, please create it manually, then copy MantisLODEditorTrial.cs there.

How to optimize all meshes in a game object?

When you select a game object, if it has any meshes, the interface of Mantis LOD Editor will automatically appear in the inspector window.

By default, help information shows below each control, you can hide them at any time if you are familiar with the interface.

Drag the slider will change the qualities of all meshes in the game object simultaneously; if you select a separate mesh in the game object, drag the slider will only change the quality of this mesh.

The game object will be reverted automatically when it lost focus, been disabled or been destroyed.

Don't forget to click the 'Save Current Mesh' bar to save the optimized meshes to disk before you leave the interface, otherwise, all your works will lose.

 

How to make LOD?

Make sure you have optimized the game object and saved the optimized meshes with different qualities to disk.

To make LOD, you need to clone several instances of the game object, then select an instance. The mesh filter or the skinned mesh renderer has a mesh property, you can see it in the inspector window, carefully drag one of the saved mesh with a certain quality from the project window to this mesh property in the inspector window, the instance will automatically become a LOD version of that quality.

Repeat the above operation, until each instance has different quality, if you don't know how to operate, please watch the tutorial videos to learn.

You may drag the LOD version to the project window to save it as a prefab for future use.

 

How to use the LOD?

After you have made LOD for a game object, you'd better save it as a prefab. Unity3d has excellent documents of how to make and use built-in LOD system in Unity Pro.

If you are using a free version of Unity, you can switch the LOD versions by distance yourself via a piece of script.

Here's the code(C#):

float distance = Vector3.Distance(Camera.main.transform.position, gameObject.transform.position);

Here's the code(Unity Script):

var distance = Vector3.Distance(Camera.main.transform.position, gameObject.transform.position);

After you have calculated the distance to the main camera, you can easily switch LOD versions by enable or disable their renderers by script.

 

Introduction of Mantis LOD Editor - Standalone Version new

Mantis LOD Editor - Standalone Version are multiple platform standalone applications which help you to simplify your 3d models while remaining all animation/rigs, it can simplify huge 3d models as you wish!

 

How to install?

It’s green software, no need to install, just unzip the package anywhere.

 

How to run?

It can run on Windows(XP/7/8) and OSX(10.8/10.9), to run the standalone version, your machine must full support OpenGL 2.0, if you are using a virtual machine, we recommend VirtualBox, their OpenGL driver for Windows 7 has been approved okay to run the standalone version.

 

On Windows, go to the Win32 subdirectory, double click MantisLODEditor.exe to launch.

 

On Mac OS X, go to the Mac subdirectory, double click MantisLODEditor.app to launch.

 

The trial version has full functions except that you cannot save to disk.

 

Which 3d format does it support?

It can import FBX, DAE, OBJ, DXF and 3DS files and export FBX, DAE, OBJ and DXF files, we recommend FBX format.

 

Both static and skinned 3d models are supported, if you have troubles with some formats, please convert your 3d model to FBX format with Autodesk’s free converting tool.

 

How can I specify the file format when saving with the standalone version?

You may select specific format when saving your model, the extension will change automaticly, this is a new feature of version 1.1.

 

Can I simplify multiple parts of 3d model with different qualities?

Just select the parts you want to simplify with certain quality, then drag the slide bar or type the quality number to simplify them, repeat untill you are satisfied with all parts, this is a new feature of version 1.1.

 

If you select nothing, simplification will apply to all parts.
